## 🚀 Featured Projects

### 🔍 [Fake News Detector](https://github.com/Nipunbhal/fake-news-detector)
> NLP model that detects fake news headlines with **99.21% accuracy**
- Built binary text classifier using TF-IDF + Logistic Regression
- Identified and fixed **data leakage** (Reuters agency tags)
- Trained on 44,000+ real and fake news articles
- `Python` `scikit-learn` `NLTK` `Matplotlib`

### 📚 [Philosopher Similarity Analysis](https://github.com/Nipunbhal/philosopher-similarity-analysis)
> Which philosophers think most alike? Analyzed 6 great thinkers using NLP
- Applied TF-IDF vectorization and cosine similarity
- Visualized results as heatmaps, word clouds and network graphs
- `Python` `scikit-learn` `NLTK` `NetworkX` `WordCloud`
